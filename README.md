نکته بسیار مهم:
دوستان گرامی در این پروژه باید ماژول nodemcu و لب تاپ یا گوشی که کنترل کننده هستند تماما به یک شبکه وای فای متصل باشند.(شبکه داخلی)

راه اندازی بردnodemcuو کنترل ledبا وب سرور و نرم افزار آردوینو :
وسایل موردنیاز:
بردبرد آموزشی
یک عددled
مقاومت 220اهم
یک بردnode mcu
